PyMC objects¶

Here we document the basic PyMC objects.

class pymc.PyMCObjects.Node(doc, name, parents, cache_depth, verbose=0)¶

The base class for Stochastic, Deterministic and Potential.

Parameters:
doc : string
The docstring for this node.
name : string
The name of this node.
parents : dictionary
A dictionary containing the parents of this node.
cache_depth : integer
An integer indicating how many of this node’s value computations should be ‘memorized’.
verbose (optional) : integer
Level of output verbosity: 0=none, 1=low, 2=medium, 3=high

See also

Stochastic
The class defining random variables, or unknown parameters.
Deterministic
The class defining deterministic values, ie the result of a function.
Potential
An arbitrary log-probability term to multiply into the joint distribution.
Variable
The base class for Stochastics and Deterministics.
